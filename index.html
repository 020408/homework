• git init - Ініціалізує новий репозиторій Git
  • git add - Додає файли до індексу для наступного коміту
  • git commit - Фіксує зміни, додані до індексу
  • git status - Переглядає стан змін у робочому каталозі та індексі
  • git log - Переглядає історію комітів
  • git branch - Створює нову гілку
  • git checkout - Перемикається на іншу гілку
  • git merge - Зливає обрану гілку з поточною
  • git rm - Видаляє файл з робочого каталогу та індексу
  • git remote add - Додає віддалений репозиторій
